# Game Folders

Each of the 20 games has its own folder:

## Category 1: Classics (5 games)
- `ludo/` — India's most-loved board game
- `chess/` — Full chess with AI
- `sudoku/` — Infinite puzzles with daily challenges
- `carrom/` — Realistic physics strike game
- `snake-ladders/` — Classic luck-based race

## Category 2: Theme Games (5 games)
- `mythology-quest/` — Snake & Ladders with deity stories
- `stock-market-ludo/` — Financial literacy + board game
- `spell-duel/` — Word battle on hex grid
- `raga-rhythm/` — Indian music memory game
- `trade-route/` — Catan-lite resource trading

## Category 3: Country-Specific (5 games)
- `pachisi-royal/` — Authentic Indian Ludo (Mughal era)
- `mancala-world/` — 6 African variants
- `diwali-dash/` — Festival race game (seasonal)
- `samurai-grid/` — Go simplified (Japan-inspired)
- `carnival-brasil/` — Samba float-building strategy

## Category 4: Age-Specific (5 games)
- `tiny-ludo/` — Age 3-5 (toddlers)
- `times-table/` — Age 8-12 (maths battles)
- `pixel-quest/` — Age 10-15 (logic puzzles)
- `debate-dice/` — Age 15-20 (social arguments)
- `retire-right/` — Age 25-45 (life simulation)

Each folder will contain:
- `index.html` — Game main file
- `game.js` — Game logic
- `ai.js` — Game-specific AI opponent
- `styles.css` — Game-specific styles
