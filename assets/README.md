# Assets Folder

Contains all visual and media assets for the studio:

## Subfolders:
- `thumbnails/` — Game thumbnails for promo panel (e.g., `ludo.png`)
- `logos/` — Microdocx studio logo, favicon, icon sets
- `screenshots/` — App Store screenshots (6 per game)
- `backgrounds/` — UI backgrounds, textures
- `icons/` — UI icons, achievement badges
- `music/` — Background music, sound effects (if applicable)

## Image Naming Convention:
- Game thumbnails: `[game-id].png` (e.g., `ludo.png`)
- Logos: `microdocx-logo-[variant].png`
- Screenshots: `[game-id]-screenshot-[number].png` (1-6)

## Sizes:
- Thumbnails: 200×200px (PNG)
- Logo: 1024×1024px (PNG, with transparency)
- Screenshots: 1080×2340px (PNG, for mobile)
- Favicon: 512×512px (PNG)

## Adding Images:
Currently, the folder is empty. Add images via GitHub's file upload or by pushing locally after cloning.
